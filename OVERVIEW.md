# AI Task Hub — 使用說明

> 這份說明寫給**想直接開始用**的人。照著做就能上手，技術細節一概不談。

---

## 一、這個工具能做什麼？

你平常同時開好幾個 AI 在幹活（幾個視窗的 Claude Code、幾個帳號的 ChatGPT 在產圖），
最煩的就是要一直切視窗去看「好了沒」。這個工具幫你把這件事反過來——**換 AI 來通知你**：

1. **AI 做完了** → 右上角跳通知＋清脆「叮」一聲，告訴你是哪個專案、做了什麼、花多久
2. **AI 卡住等你決定**（要你按允許、回答問題）→ 跳通知＋低沉「蹦」一聲
3. 隨時想看全局 → 開總覽頁面，所有 AI 任務的狀態一目瞭然
4. **ChatGPT 產完圖自動存檔**（選用）→ 生成的圖片（不管幾張）自動下載到你指定的資料夾，不用再一張張右鍵另存

> 📌 全程在**你自己的電腦**上運作，任何資料都不會上傳到網路。

---

## 二、通知怎麼看（聽）？

| 通知 | 聲音 | 意思 | 你該做什麼 |
|------|------|------|-----------|
| 🟢 已完成 | 叮（清脆） | 那個 AI 做完了 | 有空過去驗收 |
| 🟡 待決定 | 蹦（低沉） | 它卡住在等你拍板 | **盡快過去處理**，不然它會一直等 |

通知上會寫**哪個專案**（例如 `task-hub`）、**做的是哪件事**（你交代的那句話）、**花了多久**。
純粹「等你打下一句」的閒置狀態**不會**吵你——只有真的需要你的時候才響。

> 💡 想讓通知**停在螢幕上不消失**？系統設定 → 通知 → Script Editor → 樣式改成「提示」。

## 三、總覽頁面

瀏覽器開 **<http://localhost:9999>**，所有任務即時顯示（每秒更新）：

- **一列 = 一個 AI 任務**，欄位有：狀態燈、專案名、標題、正在處理的指令、
  它最後回了什麼、待辦進度、用的模型、花的 tokens、時間
- **🟡 待決定的列會整列亮橘底**，掃一眼就知道誰在等你
- 列的位置固定不亂跳；有變化時會**閃一下**提醒

常用操作：

| 想做什麼 | 怎麼做 |
|----------|--------|
| 換排序 | 點「標識／起始時間／最後更新」欄位標題，再點一次反向 |
| 自己排順序 | 直接**拖曳**任何一列上下移動 |
| 重要任務固定在最上面 | 點該列最左邊的 **📌** |
| 不想看的沉到最下面 | 點該列的 **⬇️** |
| 看被截斷的文字 | 滑鼠移上去就有完整內容 |

### ⧉ 懸浮視窗

點頁面標題旁的「**⧉ 懸浮視窗**」按鈕，會彈出一個**永遠壓在所有視窗上面**的小窗，
只顯示最重要的五欄。你在任何 app 裡工作，眼角都能看到 AI 們的狀態。
可以拖到任何角落、隨意縮放（縮太窄時會自動精簡顯示）。

> ⚠️ 小窗隸屬於總覽頁的分頁——分頁關掉小窗就跟著關。

## 四、監控 ChatGPT 要多做一步

Claude Code 的監控是全自動的，**ChatGPT 需要在每個 Chrome profile 裝一次小套件**：

1. 該 profile 開 `chrome://extensions` → 右上打開「**開發人員模式**」
2. 點「**載入未封裝項目**」→ 選 `~/projects/task-hub/chrome-extension` 資料夾
3. 在套件的「詳細資料 → 擴充功能選項」幫這個 profile **取個名字**（例如「帳號A」）→ 儲存
4. 重新整理 chatgpt.com 的分頁，完成

之後這個帳號在 ChatGPT 上產圖、跑長任務，完成時你就會收到掛著「帳號A」的通知。
Brave、Edge 等瀏覽器也可以，步驟一樣（Firefox 不支援）。

### 圖片自動下載（選用）

想讓 ChatGPT 產的圖自動存進電腦，在上面第 3 步的同一個設定頁多填一格：

1. 「**圖片自動下載的儲存資料夾**」填路徑，例如 `~/Desktop/chatgpt-images`（`~` 代表你的家目錄，填 `/Users/...` 完整路徑也可以）→ 儲存
2. 完成。之後每次產圖結束，**那一輪產的所有圖片**（一張或多張）會自動存進去，
   並跳「🖼️ 圖片已儲存」通知；檔名會帶對話標題和時間，不會互相覆蓋

不想用了就把那格清空再儲存。每個 profile 可以設不同資料夾（例如照帳號分開存）。

## 五、常見問題

**Q：完全沒收到通知？**
A：瀏覽器開 <http://localhost:9999/health>，有回 `{"ok":true}` 代表服務活著；
沒有的話重開機一次（服務會自動啟動）。還是不行就看 README 的除錯章節。

**Q：某個 Claude Code 視窗好像沒被追蹤？**
A：在那個視窗裡打一次 `/hooks` 再關掉，或把視窗重開。

**Q：ChatGPT 那邊沒反應？**
A：最常見原因是裝完套件後**忘了重新整理 chatgpt.com 分頁**。再不行，
到 `chrome://extensions` 按套件卡片上的 ↻ 重載一次。

**Q：圖片沒有自動下載？**
A：依序檢查：① 擴充功能選項裡的資料夾有沒有填、路徑是不是 `/` 或 `~/` 開頭（存的時候填錯格式會直接被擋下提示）；
② 服務活著嗎（開 <http://localhost:9999/health>）；③ 改過設定或更新過套件後，
要到 `chrome://extensions` 按 ↻ 重載、並重新整理 chatgpt.com 分頁。

**Q：通知聲音想換？**
A：可以，14 種內建音效任選，請看 README 的說明（改一行設定）。

**Q：電腦重開機後還會動嗎？**
A：會，服務設定成開機自動啟動，平常感覺不到它存在。
